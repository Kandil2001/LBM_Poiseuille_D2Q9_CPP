# Code of Conduct

## Our standard

This project should remain a respectful and constructive space for learning, CFD discussion, and open technical improvement.

Examples of positive behavior include:

- using welcoming and inclusive language
- giving constructive technical feedback
- focusing on the code, method, data, and documentation
- being respectful when discussing numerical choices or limitations
- helping others reproduce results

Examples of unacceptable behavior include:

- harassment, insults, or discriminatory language
- personal attacks
- deliberate intimidation or disruption
- publishing private information without permission
- disrespectful or unprofessional communication

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project-related communication.

## Enforcement

The maintainer may remove inappropriate comments, close issues, reject pull requests, or block participants who do not follow this code of conduct.

For serious concerns, contact the maintainer privately using the contact information in `SECURITY.md`.
