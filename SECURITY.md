# Security Policy

## Supported versions

This repository is an educational CFD validation project. The `main` branch is the active version.

| Version | Supported |
|---|---|
| `main` | Yes |
| older commits or forks | No official support |

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security-related reports.

If you find a vulnerability or a serious misuse risk, please report it privately by contacting the maintainer at:

- Email: a.akandil@outlook.com
- GitHub: https://github.com/Kandil2001

Please include:

- a short description of the issue
- steps to reproduce it, if applicable
- the affected file or function
- any suggested mitigation

For normal bugs, numerical issues, documentation problems, or feature ideas, please use the GitHub issue templates instead.
